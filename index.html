关于本站的网络线路信息

本站同时部署了2个可见的静态页面托管服务与一个 境外TencentCDN服务，分别为：

  1. 境外TencentCDN(frez79.site)（GithubPages回源 速度最快但会有页面更新延迟）
  2. Cloudflare Pages（无页面更新延迟 少几率被墙 概率出现的人机验证会拖慢下载反控软件的时间）
  3. Vercel（无页面更新延迟 速度与第一者持平 快但被墙几率大）
（首选 境外TencentCDN 请根据自己所在学校机房的网络情况选择使用）

关于本站的快速跳转功能

为了更方便/快速的下载反控软件 本站提供了快速跳转功能 格式如下：
[站点URL]?[软件ID]
例如您正以v.frez79.site访问本页面并想快速跳转到My Computer Class(ID:mcc)的网盘分流下载页面：
  v.frez79.site?mcc
或者您正以frez79.site访问本页面并想快速跳转到MythwareToolkit(ID:mt)的网盘分流下载页面：
  frez79.site?mt

ID位于标题起始处至字符|处，例如CFWorker HTTP Proxy的标题如下所示：
cfwp|CFWorker HTTP Proxy(x86):https://proxy.frez79.site/
则开头的cfwp就是ID

关于直链下载对快速跳转的影响

在对应项目含有直链时 快速跳转会默认跳转到直链链接进行下载 以便加快速度 提升体验
您可以通过在原快速下载的URL后加上 ?pu 以强制跳转到网盘链接
（PU：即PanUrl）
直链功能不会影响到其他项目，只有含有直链的项目才会跳转到直链链接
