关于本站的网络线路信息

本站同时部署了 3个静态页面托管服务，分别为：

  1. GithubPages（速度与CF不相上下）
  2. Cloudflare Pages（概率出现的人机验证会拖慢下载反控软件的时间）
  3. Vercel（速度最快但被墙几率大）
（三者均有几率被墙 请根据自己所在学校机房的网络情况选择使用）

关于本站的快速跳转功能

为了更方便/快速的下载反控软件 本站提供了快速跳转功能 格式如下：
[站点URL]?[软件ID]
例如您正以v.frez79.site访问本页面并想快速跳转到My Computer Class(ID:mcc)的网盘分流下载页面：
  v.frez79.site?mcc
或者您正以frez79.site访问本页面并想快速跳转到MythwareToolkit(ID:mt)的网盘分流下载页面：
  frez79.site?mt

ID位于标题起始处至字符|处，例如CFWorker HTTP Proxy的标题如下所示：
cfwp|CFWorker HTTP Proxy(x86):https://proxy.frez79.site/
则开头的cfwp就是ID
